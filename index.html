What Is Scale

According to the text book "Scales are functions that map from an input domain to an output range.". Therefore, it means taking a set of data and using the a function to map out the data into a new set.

Benefits

A scale can be used, which can incompose the data without the graph becoming too vast to be vizualized. Graphs with a lot of data will become too large for the audience to understand and effected by outliers. Moreover, a scale could create a proportional graph. Furthermore, a function for the data set can convert it into a new array of data and display it on a graph. Any data set can be altered to find new needed outputs.
